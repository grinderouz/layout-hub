GrinderouzDB - Clash Of Clan Base Layouts
The ultimate community-driven hub for Clash of Clans base layouts. Finding the perfect layout shouldn't be a chore. GrinderouzDB is a lightweight, open-source web application designed to help players discover, share, and instantly import the most effective base designs for every Town Hall and Builder Hall level.

✨ Key Features
Direct Import: Copy base layouts into your game with a single click using official link.clashofclans.com integration.
Smart Filtering: Sort by Town Hall (TH3–TH18),
Community Rated: Upload your own base layouts directly through our discord!
Mobile Optimized: A fast, responsive UI built for Clashers on the go.

🛠️ Code
Frontend: React / Next.js (Tailwind CSS)
